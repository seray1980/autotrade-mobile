## 方法3：手动创建（最简单）⭐⭐⭐

### 步骤1：打开页面

访问链接（或运行快速启动.bat 选择 [2]）：
https://github.com/settings/tokens

如果未登录：
- Username: seray1980
- Password: [你的 GitHub 密码]

### 步骤2：创建 Token

1. 点击右侧的："Generate new token (classic)"

### 步骤3：填写表单

在打开的页面中填写：

1. **Note**（备注）：
   输入：autotrade-mobile

2. **Expiration**（过期时间）：
   选择：No expiration（永不过期）

3. **Select scopes**（权限）：
   找到 "repo" 选项
   勾选它（会自动勾选所有子选项）
   其他选项不要勾选

### 步骤4：生成

滚动到页面底部
点击："Generate token" 按钮

### 步骤5：复制 Token（最重要！）⚠️

生成成功后，页面顶部会显示 Token：

ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

**立即复制！**
- 点击 Token 右侧的复制图标（📋）
- 或手动选择整个 Token，按 Ctrl+C

**保存 Token：**
- 打开记事本
- 粘贴 Token
- 保存到桌面：github_token.txt

⚠️ Token 只显示一次！刷新页面就看不到了！

## 常见问题

Q: Token 页面打开后，我该怎么做？
A: 按照上面的"方法3：手动创建"步骤操作

Q: 找不到 "Generate new token (classic)" 按钮？
A: 在页面向右看，应该有一个蓝色按钮

Q: Note 是什么？
A: Token 的名称，随便填，用于识别

Q: Expiration 该选什么？
A: 推荐 No expiration（永不过期），或选 90 days

Q: Select scopes 是什么？
A: 选择 Token 的权限，必须勾选 repo

Q: repo 勾选后，会有很多子选项？
A: 对，正常！让它自动勾选所有子选项

Q: Token 复制后怎么保存？
A: 保存到记事本，文件名：github_token.txt

Q: Token 输入框不显示我粘贴的内容？
A: 正常！Git 的密码框不显示内容，直接粘贴即可
