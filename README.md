## 🚀 立即开始（3 步完成）

### 步骤 1：安装 Git

双击运行：**`快速启动.bat`**

选择 `[1] 打开 Git 下载页面`

下载并安装 Git，安装时选择：
```
"Git from the command line and also from 3rd-party software"
```

---

### 步骤 2：创建 GitHub Token

双击运行：**`快速启动.bat`**

选择 `[2] 打开 GitHub Token 创建页面`

按照提示创建 Token 并**复制保存**（只显示一次！）

---

### 步骤 3：创建仓库并推送

**3.1 双击运行：`快速启动.bat`**

**3.2 选择 `[3] 打开 GitHub 新建仓库页面`**

创建仓库：
- Repository name：`autotrade-mobile`
- 选择 **Public**（公开）
- **不要勾选** 任何初始化选项

**3.3 选择 `[4] 运行 Git 部署脚本`**

脚本会自动完成所有操作，推送时输入：
- **Username**: `seray1980`
- **Password**: 粘贴你的 Token

---

### 步骤 4：等待构建完成

双击运行：**`快速启动.bat`**

选择 `[5] 打开 GitHub Actions 页面`

等待 15-30 分钟，直到看到绿色勾 ✓

---

### 步骤 5：下载 APK

在 Actions 页面点击构建记录，下载 Artifacts 中的 APK

---

### 步骤 6：安装到手机

将 APK 传输到手机并安装
